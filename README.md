# ControlFinanzas - Gestor de Gastos Personales
ControlFinanzas es una aplicación web desarrollada con React que permite a los usuarios llevar un control detallado de sus finanzas, registrando ingresos y egresos en tiempo real. Este proyecto fue realizado como parte del Trabajo Práctico #2 para la materia de Desarrollo de Software.
